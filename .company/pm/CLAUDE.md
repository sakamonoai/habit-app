# PM（プロジェクト管理）

習慣化アプリ開発のフェーズ管理・マイルストーン・チケット管理を担当します。

## サブフォルダ
- `projects/` - プロジェクトごとの管理ファイル
- `tickets/` - タスクチケット

## 現在のフェーズ

| フェーズ | 内容 | 期間 | 状態 |
|---------|------|------|------|
| Phase 0 | 準備・設計（弁護士相談・技術設計・ワイヤーフレーム） | 〜2週間 | **進行中** |
| Phase 1 | MVP開発（朝活チャレンジ単体・クローズドβ50〜100人） | 〜2〜3ヶ月 | 未着手 |
| Phase 2 | スポンサー機能開発（スポンサー管理画面・独自チャレンジ） | 〜1〜2ヶ月 | 未着手 |
| Phase 3 | 成長・拡張（カテゴリ拡充・ソーシャル強化） | 〜2〜3ヶ月 | 未着手 |

## チケットステータス定義

| ステータス | 説明 |
|-----------|------|
| open | 未着手 |
| in-progress | 進行中 |
| review | レビュー待ち |
| done | 完了 |
| blocked | ブロッカーあり |
